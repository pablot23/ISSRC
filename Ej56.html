Crea una variable llamada `calificacion` y asígnale un valor. Utiliza una estructura condicional `if...else if...else` para mostrar en la consola si la calificación es "Excelente," "Aprobado," o "Reprobado"

El codigo es el siguiente:

 
            
        let calificacion = 100;
        if (calificacion==100) {
        console.log("excelente")
        } 
        else if (calificacion>60) {
        console.log("aprobo");
        } 
        else {
        console.log("no aprobo");
        }
