Página web responsiva utilizando CSS media queries

El codigo es el siguiente:

 *{
        margin:0;
        padding:0;
        }
        body{
        background:#323A46;
        font-family:Century Gothic;
        }
        .contenedor{
        width:80%;
        max-width:1000px;
        margin:0 auto;
        /*background:blue;*/
        overflow:hidden;
        }
        header{
        background:#EB4A33;
        margin:20px 0;
        }
        header h1{
        color:#fff;
        padding:20px 0;
        text-align:center;
        }
        .main{
        width:70%;
        background:#fff;
        padding:20px;
        float: left;
        box-sizing:border-box;
        }
        .main img{
        width:100%;
        height:auto;
        }
        aside{
        width:30%;
        padding:20px;
        box-sizing:border-box;
        background:#ccc;
        float:left;
        }
        footer{
        clear:both;
        float:left;
        margin-top:20px;
        box-sizing:border-box;
        width:100%;
        padding:20px;
        color:#fff;
        background:#000;
        text-align:center;
        }
        @media screen and (max-width: 800px){
            .main{width:100%;
        }
            .aside{width:100%;
        }
            body{background:#ff8000;
        }
        }
        @media screen and (max-width: 400px){
            aside{display:none;
             
        }
