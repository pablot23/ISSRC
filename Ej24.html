Estiliza un enlace cambiando el color y el efecto al pasar el mouse (hover)

El codigo es el siguiente:

 a:link {
        color: blueviolet;
      }
      
      a:visited {
        color: green;
      }
      
      a:hover {
        border-bottom: 1px solid;
        background: aqua;
      }
      
      a:active {
        background: #265301;
        color: #cdfeaa;
      }

Nota: siempre tiene que ir en orden "link-visited-hover-active"
