Escribe una función llamada `esPar` que tome un número como argumento y retorne `true` si es par o `false` si es impar. Luego, muestra el resultado en la consola.

El codigo es el siguiente:

 
            
            function esPar(numero) {
  
            return (numero % 2) == 0;
                
            }
                
            let numero = 8; 
                
            let resultado = esPar(numero);
                
            console.log(resultado);
